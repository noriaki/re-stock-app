## ディレクトリ配置規則

```
app/*                 # アプリケーション
components/*          # コンポーネント
lib/                  # 一般ユーティリティ
types/                # 型定義
public/               # 静的ファイル
.cline/               # Cline用のディレクトリ
.cline/rules/         # .clinerulesの元となる振る舞い･ルール
.cline/memory/        # メモリバンク
.cline/memory-bank/   # 追加のコンテキストや知識情報
docs/                 # ユーザ向けドキュメント置き場
```
